EHRforEquips

A management software for equips of freelance doctors.

Project Overview

EHRforEquips is a multiplatform, portable software designed for freelance doctors working in multiple structures. It follows a plugin architecture, allowing doctors to customize their workflow by adding preferred plugins. The software is compatible with Linux, Windows, and MacOS.

Features

  • Plugin architecture for customization
  • Multiplatform support (Linux, Windows, MacOS)
  • Patient and visit management
  • Easy plugin integration

Technologies Used

  • Kotlin
  • Java
  • IntelliJ IDEA
