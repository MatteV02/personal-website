kotlin-core

A repository of exercises in Kotlin, adapted from nbicocchi/java-core.

Project Overview

This repository is an adaptation of the exercises provided in nbicocchi/java-core for the Kotlin programming language. It contains exercises covering basics, object-oriented design, data structures, generics, functional programming, exceptions, I/O, and multi-threading.

Features

  • Exercises for Kotlin basics
  • Object-oriented design exercises
  • Data structures and collections
  • Generic data structures
  • Functional programming exercises
  • Exception handling
  • I/O framework exercises
  • Multi-threading exercises

Technologies Used

  • Kotlin
  • Gradle
